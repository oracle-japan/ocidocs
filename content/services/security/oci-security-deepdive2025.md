2024年10月29日開催の、Oracle Cloud Infrastructure(OCI) Technical Deep Dive にて使用した資料です。

本資料では、基幹システム移行で欠かせないセキュリティ設計と運用監視の要点を、OCIのベストプラクティスとあわせて解説します。
特に今回は Oracle Databaseの“守り方”と“見守り方” にフォーカスします。

**主なトピック**
- DBの守り方：暗号化、アクセス制御、監査、データ保護(例：Data Safe 等)
- 監視と運用：メトリクス／ログ統合、ログ分析、アラーム、自動修復ワークフロー
- 事故を起こさない運用：変更管理、バックアップ／DR、インシデント対応の型
